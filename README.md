## HR Data Analytics Dashboard (MS Excel And TABLEAU)

**Project Overview:**

This project focuses on analyzing HR data to identify key factors contributing to employee attrition and departmental performance. Using MS Excel, I processed raw workforce data to create an interactive dashboard that provides actionable insights for HR management.

**Key Metrics Tracked:**

Attrition Rate: Calculated the percentage of employees leaving the company.
Department Distribution: Breakdown of headcount across Sales, R&D, and HR.
Salary Trends: Analysis of average income vs. job role and experience.
Employee Satisfaction: Correlation between environment satisfaction and retention.

**Technical Workflow:**

Data Cleaning: Handled missing values, removed duplicates, and standardized job titles using Power Query.
Data Processing: Utilized VLOOKUP, XLOOKUP, and IF statements to categorize salary tiers and tenure.
Analysis: Created Pivot Tables to aggregate data by department, gender, and education level.
Visualization: Built an interactive dashboard using Slicers (Region, Department, Year) and dynamic charts.
Insights & Recommendations:
Insight: High attrition was noted in the Sales department among employees with 2–3 years of tenure.
Recommendation: Suggested a mid-career retention program and revised bonus structures for high-pressure roles.
